Модуль 2: Управляющие конструкции

Условные операторы

Условные операторы позволяют выполнять код в зависимости от заданных условий. В Python используются операторы if, elif и else.

Оператор if

Оператор if позволяет выполнить блок кода, если условие истинно:

                        
                            x = 10
                            if x > 5:
                                print("x больше 5")

Оператор if-else

Оператор if-else позволяет выполнить один блок кода, если условие истинно, и другой блок, если условие ложно:

                        
                            x = 3
                            if x > 5:
                                print("x больше 5")
                            else:
                                print("x не больше 5")

Циклы

Циклы позволяют выполнять блок кода несколько раз. В Python используются циклы for и while.

Цикл for

Цикл for используется для итерации по последовательности (списку, кортежу, строке и т.д.):

                    
                        for i in range(5):
                           print(i)

Цикл while

Цикл while выполняется до тех пор, пока условие истинно:

                        
                            count = 0
                            while count < 5:
                               print(count)
                               count += 1

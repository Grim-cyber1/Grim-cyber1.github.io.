Модуль 4: Функции

Определение функций

Функции в Python - это блоки кода, которые выполняют определенную задачу. Они помогают разбить программу на модули и повторно использовать код.

                        
                            def greet(name):
                                print("Hello, " + name + "!")
                           greet("Alice")

Аргументы функций

Функции могут принимать аргументы, которые передаются в функцию при ее вызове.

                        
                            def add(x, y):
                               return x + y
                           result = add(5, 3)
                           print(result)

Возврат значений

Функции могут возвращать значение с помощью оператора return.

                        
                           def multiply(a, b):
                              return a * b
                           result = multiply(4, 6)
                           print(result)

Область видимости переменных

Переменные, объявленные внутри функции, имеют локальную область видимости и не видны за пределами функции.

                       
                            def my_function():
                                x = 10
                                print(x)
                           my_function()

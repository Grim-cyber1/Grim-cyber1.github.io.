Модуль 1: Основы Python

Введение в Python

Python - это мощный и простой в изучении язык программирования. Он подходит как для новичков, так и для опытных разработчиков.

Основные концепции

  • Переменные: Хранят данные (числа, текст и т.д.)
  • Типы данных: Определяют, какого рода данные хранятся
  • Операторы: Символы для выполнения операций
  • Ввод/вывод: Позволяют общаться с программой

Пример простого вывода:

                       print("Hello, Python!")

Это выведет на экран "Hello, Python!".
